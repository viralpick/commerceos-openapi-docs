# Getting started with COS

COS(Commerce Operating System) 플랫폼은 데이터를 활용해 실제 비즈니스 문제를 해결할 수 있도록 설계되었습니다. COS 플랫폼은 다음과 같은 조직과 사용자가 활용합니다:

- **다양한 조직**: 이커머스 기업, 브랜드사, 유통사, 마켓플레이스 운영사
- **다양한 사용자**: 데이터 엔지니어, 데이터 분석가, MD, 마케터, 운영 담당자

## Platform at a glance

COS는 데이터, 지식, 실행을 하나의 흐름으로 연결하는 **Agent Operating System**입니다.

| 모듈 | 역할 |
|------|------|
| **Pipeline Builder** | DB, API, 파일을 연결하고 전처리/정규화 후 온톨로지 DB로 적재 |
| **Ontology** | 데이터에 의미와 관계를 부여해 AI가 이해할 수 있도록 구조화 |
| **Workflow Builder** | 멀티 에이전트 기반 워크플로우 구성 및 실행 |
| **View Generator** | 온톨로지 기반 대시보드 자동 생성 |

> 데이터 연결부터 액션까지, 모든 과정을 **COS** 하나의 플랫폼에서 해결
