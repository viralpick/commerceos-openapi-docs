# Open API

AOS Open API를 사용하면 외부 시스템에서 AOS 플랫폼의 데이터와 기능에 프로그래밍 방식으로 접근할 수 있습니다.

---

## API Key 발급

AOS Open API를 사용하기 위해서는 먼저 API Key를 발급받아야 합니다.

### 1. 내 계정 페이지 접근

AOS 플랫폼에 로그인한 후, 좌측 메뉴에서 **내 계정**을 클릭합니다.

### 2. API Key 섹션 확인

내 계정 페이지 하단에 **API Key** 섹션이 있습니다. 이 섹션에서 API Key를 관리할 수 있습니다.

![API Key 관리 화면](/api-key-management.png)

### 3. 새 API Key 발급

1. **새 API Key 발급** 버튼을 클릭합니다.
2. API Key의 **이름**과 **설명**을 입력합니다.
3. **발급** 버튼을 클릭하면 새로운 API Key가 생성됩니다.

> **주의**: API Key는 생성 시점에만 전체 값을 확인할 수 있습니다. 이후에는 마스킹된 값만 표시되므로, 생성 즉시 안전한 곳에 저장하세요.

### 4. API Key 관리

발급된 API Key는 테이블에서 다음 정보를 확인할 수 있습니다:

| 항목       | 설명                                        |
| ---------- | ------------------------------------------- |
| **이름**   | API Key의 식별 이름                         |
| **키**     | 마스킹된 API Key 값 (예: `sk_live_XVzC***`) |
| **설명**   | API Key에 대한 설명                         |
| **생성일** | API Key 생성 날짜                           |

필요하지 않은 API Key는 우측의 삭제 버튼을 클릭하여 삭제할 수 있습니다.

## API 인증

API 요청 시 발급받은 API Key를 사용하여 인증합니다.

### Header 인증

```bash
curl -X GET "https://api.commerceos.ai/v1/endpoint" \
  -H "Authorization: Bearer sk_live_your_api_key"
```

### 인증 방식

| 방식         | Header          | 값                 |
| ------------ | --------------- | ------------------ |
| Bearer Token | `Authorization` | `Bearer {API_KEY}` |
