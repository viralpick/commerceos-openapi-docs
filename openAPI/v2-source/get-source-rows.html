Get Sources API

자사가 보유한 Source Data Row값을 불러오는 API입니다.

인증

API에 접근하기 위해서는 API Key가 필요합니다. 다음은 예시입니다:

curl -X GET https://app-api-v2.commerceos.ai/open/v1/sources/{sourceId} \
     -H "x-api-key: YOUR_API_KEY"

엔드포인트

소스 데이터 가져오기

엔드포인트: GET /open/v1/sources/{sourceId}

요청

필드타입필수 여부설명
x-api-key (헤더)string예사용자별 API Key.
sourceId (Path)string예row를 조회할 source id - get source api를 통해 조회가능합니다.

예시 요청

curl -X GET https://app-api-v2.commerceos.ai/open/v1/sources/{sourceId} \
     -H "x-api-key: YOUR_API_KEY"

응답

필드타입필수 여부설명
metameta예API에 대한 결과와 응답 메시지를 담고 있습니다.
dataarray Source예Source 객체의 배열입니다.
