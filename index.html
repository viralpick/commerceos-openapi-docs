Getting started with COS

Getting started with COS

COS(Commerce Operating System) 플랫폼은 데이터를 활용해 실제 비즈니스 문제를 해결할 수 있도록 설계되었습니다. COS 플랫폼은 다음과 같은 조직과 사용자가 활용합니다:

  • 다양한 조직: 이커머스 기업, 브랜드사, 유통사, 마켓플레이스 운영사
  • 다양한 사용자: 데이터 엔지니어, 데이터 분석가, MD, 마케터, 운영 담당자

Get access

COS 플랫폼을 시작할 준비가 되었다면:

  • Demo 신청: Enhans 웹사이트 에서 데모를 신청하세요.
  • Onboarding: Enhans 엔지니어와 함께 데이터 연결부터 첫 번째 Use Case 구현까지 진행합니다.

Understand the platform

이 문서를 효과적으로 활용하기 위한 안내입니다:

문서내용
Platform OverviewCOS의 4가지 핵심 모듈(Pipeline Builder, Ontology, Workflow Builder, View Generator) 소개
Ontology ConceptsObject, Link, Knowledge Dictionary의 개념과 Annotation 작성 기준
Workflow Builder노드 구성, Agent 입력 소스, 데이터 전달 방식
Open API Reference데이터 접근 및 관리를 위한 API 문서

Platform at a glance

COS는 데이터, 지식, 실행을 하나의 흐름으로 연결하는 Agent Operating System입니다.

모듈역할
Pipeline BuilderDB, API, 파일을 연결하고 전처리/정규화 후 온톨로지 DB로 적재
Ontology데이터에 의미와 관계를 부여해 AI가 이해할 수 있도록 구조화
Workflow Builder멀티 에이전트 기반 워크플로우 구성 및 실행
View Generator온톨로지 기반 대시보드 자동 생성

데이터 연결부터 액션까지, 모든 과정을 COS 하나의 플랫폼에서 해결

Start building your use case

플랫폼 접근 권한을 받은 후, 다음 리소스를 활용해 시작하세요:

  • Examples: Use Case별 레퍼런스 예제와 튜토리얼
  • Ontology Setup Guide: 데이터 구조화를 위한 온톨로지 설정 가이드
  • Workflow Templates: 자주 사용되는 워크플로우 템플릿

Use case examples

재고 관리수요 예측 기반 인벤토리 최적화
